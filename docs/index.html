Python Email Server
with GUI

A full-featured SMTP email server for sending real emails and testing locally

Download View on GitHub

$ git clone https://github.com/kiprutobeauttah/Email-server.git

$ cd Email-server

$ pip install -r requirements.txt

$ python main.py

✓ Email Server Started Successfully!

Features

Three Operating Modes

Local testing, direct send without auth, or authenticated SMTP via Gmail/Outlook

Send Real Emails

Deliver emails to Gmail, Yahoo, Outlook, and any email provider worldwide

File Attachments

Attach multiple files to your emails with full MIME support

Multiple Recipients

Send to multiple recipients using To, CC, and BCC fields

Inbox Management

View, clear, and refresh received emails with an intuitive interface

Email Validation

Automatic validation of email formats and SMTP connection testing

Quick Installation

# Clone the repository
git clone https://github.com/kiprutobeauttah/Email-server.git
cd Email-server

# Install dependencies
pip install -r requirements.txt

# Run the application
python main.py

Requirements

  • ✓ Python 3.7 or higher
  • ✓ tkinter (usually included with Python)
  • ✓ aiosmtpd >= 1.4.4
  • ✓ dnspython >= 2.3.0

How to Use

Server Tab

Start the local SMTP server to receive emails. Configure host and port, then click "Start Server" to begin receiving.

Send Email Tab

Choose your mode (Local, Direct, or Gmail SMTP), compose your email, add attachments, and send.

Inbox Tab

View all received emails, click to read full content, refresh the list, or clear all messages.
